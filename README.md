# Instagram Reach Analysis

## Project Overview

This project analyzes Instagram post performance using Python.

## Objectives

- Analyze engagement metrics
- Discover factors affecting impressions
- Build a prediction model
- Generate business insights

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Dataset Features

- Impressions
- Likes
- Comments
- Shares
- Saves
- Profile Visits
- Follows

## Key Insights

- Shares contribute significantly to impressions.
- Saves and Likes have a positive relationship with reach.
- Profile Visits increase with higher engagement.
- Linear Regression was used to predict impressions.
